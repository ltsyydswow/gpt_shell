# 小朋友开发的 Linux 程序

这是一个小朋友开发的程序，通过一系列巧合，我看到了一位优秀的 UP 主的视频。视频中的代码非常优秀，但是却是针对 Windows 平台开发的。于是，我决定开发一个在 Linux 平台上运行的版本。

## 特点

- 在 Linux 平台上运行，为 Linux 用户提供更好的体验。
- 基于优秀的代码进行开发，确保高质量和稳定性。
